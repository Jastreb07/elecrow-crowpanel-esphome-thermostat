# Smart Thermostat Knob Home Assistant integration

This integration has moved to its own repository:
**[Jastreb07/elecrow-crowpanel-esphome-thermostat-integration](https://github.com/Jastreb07/elecrow-crowpanel-esphome-thermostat-integration)**.

It's a HACS-installable Home Assistant integration that picks which
climate/light/cover entities show up on a Smart Thermostat Knob device
(this firmware repository) — see that repo's README for installation
(HACS or manual) and usage.

For the manual, integration-free alternative (hand-written Template-Entity
YAML), see [SETUP.md](../SETUP.md#home-assistant-setup) in this repo.
