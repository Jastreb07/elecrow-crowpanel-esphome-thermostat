# fonts/

Font assets used by the LVGL layouts.

The current firmware references the bundled Rajdhani TTF files from this
directory. Icons come from the Material Design Icons webfont, which ESPHome
downloads at build time (see the `font:` sections in `thermostat_480.yaml`
and `thermostat_240.yaml`). Add new font files here and reference them from
those `font:` sections.
