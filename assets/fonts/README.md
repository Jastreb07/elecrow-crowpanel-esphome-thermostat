# fonts/

Platzhalter für zusätzliche/eigene Schriftdateien. `thermostat.yaml`
nutzt aktuell ausschließlich in ESPHome eingebaute Montserrat-Größen
(`montserrat_14`/`_20`/`_28`/`_48`), es sind keine externen Font-Dateien
nötig. Für individuelle Schriftgrößen/-schnitte (z.B. exakt aus
SquareLine übernommene TTFs) hier ablegen und in `thermostat.yaml` unter
`font:` referenzieren.
