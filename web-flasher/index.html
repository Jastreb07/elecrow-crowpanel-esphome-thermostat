Smart Thermostat Knob

Flash the ESPHome firmware for your CrowPanel rotary display directly from Chrome or Edge over USB — no software install required. After flashing, configure Wi-Fi right here in the browser. Source and documentation: GitHub.

CrowPanel 1.28″

Round rotary display, 240×240, GC9A01A panel. Product page.

Your browser does not support Web Serial. Use Chrome or Edge on desktop. Web Serial is only available on secure (HTTPS) origins.

CrowPanel 2.1″

Round rotary display, 480×480, ST7701S panel. Product page.

Your browser does not support Web Serial. Use Chrome or Edge on desktop. Web Serial is only available on secure (HTTPS) origins.

How it works

  1. Connect the board to your computer with a USB cable.
  2. Click Connect and Install for your board size and pick the serial port in the browser prompt.
  3. Wait for the firmware to flash; the page shows progress.
  4. Once flashing finishes, the installer offers a Configure Wi-Fi step (Improv Wi-Fi) — pick your network and enter the password without leaving the browser.
  5. The device restarts, joins Wi-Fi, and shows up in Home Assistant. Continue with the Home Assistant setup.
Smart Thermostat Knob is not affiliated with Elecrow or Home Assistant. Firmware built with ESPHome, flashing powered by ESP Web Tools.
